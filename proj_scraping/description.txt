scrapy startproject proj_scraping

Создаёт директорию proj_scraping/:

scrapy.cfg - настройки проекта;
-/proj_scraping/ - Python модуль проекта;
-/items.py - классы, описывающие модель собираемых данных (перечисляют поля данных);
-/pipelines.py - используется в основном для описания пользовательских форматов сохранения результатов парсинга (позволяет задать определенные действия при открытии/закрытии паука, сохранения данных);
-/settings.py - пользовательские настройки паука;
-/spiders/ - директория, в которой хранятся файлы с классами пауков. Каждого паука принято писать в отдельном файле с именем name_spider.py.
